# tealium-react-adobe-visitor

A native module for Tealium's Adobe Visitor Service

## Installation

```sh
npm install tealium-react-adobe-visitor
```

## Usage

```js
import { multiply } from 'tealium-react-adobe-visitor';

// ...

const result = await multiply(3, 7);
```
